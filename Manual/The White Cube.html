On the Subject of The White Cube

You know that one default object that people delete a lot? Well, it wants its revenge.

This module consists of a white cube of 6 different faces, with each face containing of various different twists of a familiar module.

To disarm this module, find the face that consists of a 10 digit keypad, and input the correct 5-digit value.

All other faces do not need to be disarmed but can be used to assist with the final calculation.

This module consists of an intertwined message consisting of 3 letters. This is linked together to form a single message, with a long rest denoting the end of the sequence.

To disarm the module, decode the Morse code based on the dots and dashes it flashes, then decode the Tap code based on the number of flashes on each set. Note that the table for Tap Code is used in row, col format for the 2 flashes, and K will NOT be used in Tap Code. Concatenate the result, with the Morse sequence being first in the set to get your message.

On the Subject of Quinary Converter

This balanced base is useless! I swear someone thought he was clever enough to figure it out.

This face consists of 3 screens, the bottom screen displaying the input, the top screen displaying a value in base 5, or balanced quinary, and a side screen which will display the original value when the face is solved.

There are also 7 buttons, labeled with quinary values for inputting both balanced and standard base 5 digits, a green submit button, and a red clear button.

To disarm this face, input the correct balanced quinary if the top display is showing a number in base 5, or in base 5 otherwise. Both the balanced quinary and the base 5 values must point to the same number upon submission.

Incorrectly submitting the wrong digits in the sequence will result in a strike and clear the inputs in the process, but will NOT reset this module.

Appendix: Converting Base-5 To Balanced Quinary

To convert a base-5 value to balanced quinary, perform the following steps from right to left on the given value:

625 125 25 5 1
1 4 0 2 3
1 4 0 2+1 3-5
1 4 0 3 -2
1 4 0+1 3-5 -2
1 4 1 -2 -2
1+1 4-5 1 -2 -2
2 -1 1 -2 -2
  1. Start with the least significant digit in that set. This is the right-most digit of the number in base 5.
  2. Repeat the following procedure until there are no more significant digits to convert:
    • If the digit is a 0, 1, or 2, leave that digit alone.
    • If the digit is a 3, 4, or 5 (after adding), subtract 5 to this digit and add one to the more significant digit.
    • Go to the more significant digit and check on the top 2 statements above.
  3. This should result in your balanced quinary.
