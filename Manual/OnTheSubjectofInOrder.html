On The Subject of In Order

There will always be chaos. But now it’s up to you to remove it for now.

On the module is:

  • a left display showing a number inside a coloured shape. This changes after every *solved module.
  • a right display showing the stage. This increases by one after every *solved module.
  • four directional arrows.
  • a bottom display showing a word. This changes after every *solved module.

To solve the module, you must get a number from:

  • the number inside the shape.
  • the shape itself.
  • the colour of the shape.

Add these numbers together. Then, subtract n from the number until you get a number between 1 and 8, n being the number from the word being shown by the bottom display.

When you have a number, look at Table A to find what arrow needs to be pressed next. BUT WAIT! **Do not press any buttons yet. When all modules (that aren’t ignored) have been solved, the displays on this module will go blank. You will now be able to input the sequence of arrows. If all is correct, the module will deactivate.
